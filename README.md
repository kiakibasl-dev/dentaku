# 多機能電卓アプリ（CUI）

コマンドラインで動く電卓アプリです。
## 1.概要
Windowsのデフォルトの電卓アプリを再現しました

## 2.開発背景
### 学生時代にC++を独学
大学の必修授業でPascalを学ぶ。他言語も学びたいと思いC++を独学。
### GUI経験なし
授業、競技プログラミングはどちらもCUIしか扱わず、、、

## 3.機能
### 機能一覧
|項目|内容|
|:---|:---|
|標準の電卓|四則演算|
|進数変換|10進数を16までの進数に変換|
|為替レート|為替レートを取得し計算|
### 使い方
#### 標準の電卓
計算を続ける
#### 進数変換
10進数を任意（2～16）の進数に変換
#### 為替レート
http通信で為替レートを取得、金額入力で計算

## 4.開発環境
| 項目 | 内容 |
|:---|:---|
|エディタ|VScode|
|コンパイラ|MinGW64|
|ライブラリ|httplib.h/json.hpp|
|外部API|http://www.floatrates.com|

## 5.工夫した点
エラーハンドリングやUI設計
